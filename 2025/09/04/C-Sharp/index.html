.NET平台与框架

.NET 是“平台”，而 ASP.NET 是建立在 .NET 平台之上的“框架”

.NET 平台的主要“分支”或实现包括：

  • .NET Framework (旧)：原始的 .NET，仅限 Windows。

  • .NET Core (跨平台重生)：现代、跨平台、高性能的开源实现。

  • .NET 5/6/7/8+ (统一平台)：.NET Core 的后续版本，统一了整个 .NET 生态系统，是未来的方向。

ASP.NET 是 专门基于 .NET 平台来构建 Web 应用程序和服务的框架

ASP.NET 本身也有两个主要版本：

  • ASP.NET (基于 .NET Framework)：传统的 Web 框架，用于在 Windows 上构建 Web Forms、MVC 应用。（旧）

  • ASP.NET Core (基于 .NET Core / .NET 5+)：现代、跨平台、高性能、开源的重写版本。它是构建新 Web 项目的默认和推荐选择。
